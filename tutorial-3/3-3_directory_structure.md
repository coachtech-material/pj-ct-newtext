# Tutorial 3-3: ディレクトリ構造の理解

## 🎯 このセクションで学ぶこと

*   コンピュータのファイルシステムが、「木」のような階層構造（ツリー構造）になっていることを理解する。
*   ルートディレクトリ、ホームディレクトリ、カレントディレクトリ、親ディレクトリという、重要な4つのディレクトリの概念を理解する。
*   場所を移動するためのコマンド `cd` を使いこなし、ディレクトリ間を自由に行き来できるようになる。

---

## 導入

前のセクションで、`pwd` コマンドを使って「現在地」の住所を確認しました。その住所は、`/Users/YourName/Desktop/hello-world` のような、スラッシュ `/` で区切られた長い文字列でしたね。

これは、コンピュータの中のファイルやフォルダ（ディレクトリ）が、**階層構造（ツリー構造）**で整理されていることを示しています。ちょうど、大きな木に太い枝があり、その枝から細い枝が伸び、その先に葉っぱがついているようなイメージです。

GUI（エクスプローラーやFinder）でフォルダをダブルクリックして、中のフォルダをさらにダブルクリックしていく操作は、この木の枝をどんどん辿っていくのと同じことです。

コマンドラインを使いこなす上で、この「コンピュータの中の地図」を頭の中に描けることは、非常に重要です。自分が今、木のどの部分にいるのかを常に意識することで、コマンド操作のミスを劇的に減らすことができます。

このセクションでは、このディレクトリの階層構造を理解し、その木の中を自由自在に移動するための、最も重要なコマンド `cd` を学びます。

---

## 概念の説明

### 🔑 木の構造: ルートディレクトリと階層

コンピュータのファイルシステムは、**ルートディレクトリ (Root Directory)** と呼ばれる、たった一つの最上位のディレクトリから始まります。これは、木の「根っこ」に相当します。

*   **Mac/Linux**: ルートディレクトリは `/` （スラッシュ）という名前です。
*   **Windows**: `C:` や `D:` のように、ドライブごとにルートディレクトリ（ドライブレター）が存在します。

このルートディレクトリから、全てのファイルやディレクトリが、枝葉のように階層的にぶら下がっています。

（ここに図を挿入: `/` を頂点とし、`Users`, `bin`, `etc` などに枝分かれし、さらに `Users` から `YourName` へ、`YourName` から `Desktop` へと枝分かれしていくツリー構造の図）

`pwd` で表示された `/Users/YourName/Desktop/hello-world` という住所は、この木構造を「根っこからの道順」で表現したものです。

1.  まず根っこ `/` から出発し、
2.  `Users` という枝に進み、
3.  次に `YourName` という枝に進み、
4.  `Desktop` という枝に進んだ先にある、
5.  `hello-world` という葉っぱ（場所）

このように、スラッシュ `/` は、ディレクトリの「区切り」を意味しています。

### 🔑 重要な4つのディレクトリ

コマンドライン操作で頻繁に登場する、特別な意味を持つディレクトリが4つあります。これは必ず覚えましょう。

| 用語 | 記号 | 説明 | 例え |
| :--- | :--- | :--- | :--- |
| **ルートディレクトリ** | `/` | ファイルシステムの「根っこ」。全てのディレクトリの最上位。 | 地球 |
| **ホームディレクトリ** | `~` | ログインしているユーザー専用の「自分の部屋」。 | 自分の家 |
| **カレントディレクトリ** | `.` | 「今いる場所」。`pwd` で表示されるディレクトリ。 | 現在地 |
| **親ディレクトリ** | `..` | 「今いる場所の、一つ上の階層」。 | 今いる部屋の、一つ外の廊下 |

特に、`~` (ホーム)、`.` (今)、`..` (一つ上) の3つの記号は、この後学ぶ `cd` コマンドで、魔法のように便利に使うことができます。

---

## 詳細解説

### ⚙️ 場所を移動するコマンド: `cd` (Change Directory)

`cd` は **C**hange **D**irectory の略で、カレントディレクトリ（現在地）を、指定した別のディレクトリに移動するためのコマンドです。GUIでフォルダをダブルクリックする操作に相当します。

**基本構文:**

```bash
cd [移動先のディレクトリ名]
```

それでは、実際に `cd` コマンドを使って、ディレクトリの木の中を冒険してみましょう。VSCodeでターミナルを開き、`hello-world` フォルダにいる状態から始めてください。

### ⚙️ 1. 親ディレクトリに移動する: `cd ..`

まず、今いる `hello-world` の一つ上の階層、つまり `Desktop` に移動してみましょう。「一つ上の階層」を表す特別な記号 `..` を使います。

```bash
# 現在地は hello-world
YourName@MacBook-Air hello-world % pwd
/Users/YourName/Desktop/hello-world

# cd .. を実行
YourName@MacBook-Air hello-world % cd ..

# 移動後の現在地を確認
YourName@MacBook-Air Desktop % pwd
/Users/YourName/Desktop
```

`cd ..` を実行すると、プロンプトの表示が `hello-world` から `Desktop` に変わったのが分かりますか？ `pwd` で確認すると、現在地が一つ上の `Desktop` に移動していることが確認できます。

もう一度 `cd ..` を実行すれば、さらに一つ上の `YourName` ディレクトリに移動できます。このように、`cd ..` を繰り返すことで、木の根っこの方へどんどん登っていくことができます。

### ⚙️ 2. 子ディレクトリに移動する

今度は逆に、今いる場所から、中にあるフォルダ（子ディレクトリ）に移動してみましょう。現在地の `Desktop` から、再び `hello-world` に戻ります。

```bash
# 現在地は Desktop
YourName@MacBook-Air Desktop % ls
hello-world   OtherFolder ...

# cd hello-world を実行
YourName@MacBook-Air Desktop % cd hello-world

# 移動後の現在地を確認
YourName@MacBook-Air hello-world % pwd
/Users/YourName/Desktop/hello-world
```

`ls` で `hello-world` というフォルダがあることを確認し、`cd hello-world` を実行することで、その中に入ることができました。

### ⚙️ 3. ホームディレクトリに一瞬で戻る: `cd` または `cd ~`

木の深い階層まで移動した後、一瞬で自分の拠点である「ホームディレクトリ」に戻りたい時があります。そんな時は、引数を何も指定せずに `cd` とだけ打つか、ホームディレクトリを表す `~` を使って `cd ~` と打ちます。

```bash
# 現在地は hello-world
YourName@MacBook-Air hello-world % pwd
/Users/YourName/Desktop/hello-world

# cd を実行
YourName@MacBook-Air hello-world % cd

# 移動後の現在地を確認
YourName@MacBook-Air ~ % pwd
/Users/YourName
```

どんなに深い場所にいても、`cd` コマンド一発で、自分の部屋（ホームディレクトリ）に戻れる。これは非常に便利なテクニックです。

## ✨ まとめ

このセクションでは、コマンドライン操作の基礎となる、ディレクトリの階層構造と、その中を移動するための `cd` コマンドについて学びました。

*   **ツリー構造**: ファイルシステムは、ルートディレクトリ (`/`) を頂点とする、木の形をした階層構造になっている。
*   **4つの重要ディレクトリ**: 
    *   ルート (`/`): 根っこ
    *   ホーム (`~`): 自分の部屋
    *   カレント (`.`): 今いる場所
    *   親 (`..`): 一つ上の場所
*   **`cd` コマンド**: **C**hange **D**irectory。ディレクトリ間を移動するためのコマンド。
    *   `cd ..`: 一つ上の親ディレクトリに移動する。
    *   `cd [子ディレクトリ名]`: 指定した子ディレクトリに移動する。
    *   `cd` または `cd ~`: ホームディレクトリに一瞬で移動する。

`pwd` (どこにいるか確認), `ls` (周りに何があるか確認), `cd` (別の場所に移動)。この3つのコマンドを使いこなせるようになれば、あなたはもう、コマンドラインの世界の地図を手に入れ、自由に歩き回る冒険者の第一歩を踏み出したことになります。
