# Tutorial 2-5: 開発環境の動作確認

## 🎯 このセクションで学ぶこと

*   VSCodeを使って、最初のHTMLファイルを作成し、編集する。
*   作成したHTMLファイルを、Google Chromeで表示する。
*   「コードを書き、ブラウザで確認する」という、Web開発の最も基本的な流れを体験する。

---

## 導入

お疲れ様でした！これで、開発環境という「キッチン」に、基本的な道具が一通り揃いました。

*   **お皿とフォーク**: Google Chrome
*   **包丁とまな板**: Visual Studio Code

本格的な料理（Laravelを使った開発）に入る前に、まずはこの新しいキッチンがちゃんと使えるかどうか、簡単な料理で試してみましょう。火を使わない、サラダのような簡単な料理です。

このセクションでは、「**VSCodeで簡単なWebページを作り、それをChromeで表示する**」という、Web開発の最も基本的なワークフローを体験します。これは、あなたがこれから何百回、何千回と繰り返すことになる、基本中の基本の動作です。

この簡単な成功体験が、「プログラミングって、自分で作ったものが形になるんだ！」という実感と、次へ進むための大きな自信に繋がります。

## 詳細解説

それでは、記念すべき最初のプログラミングを始めましょう！

### ⚙️ 1. プロジェクトフォルダの作成

まず、これから作るファイルを保存しておくための「専用の引き出し」、すなわち**プロジェクトフォルダ**を作成します。

1.  PCの分かりやすい場所（デスクトップなど）に、新しいフォルダを作成します。
2.  フォルダ名は、半角英数で `hello-world` としましょう。プログラミングの世界では、フォルダ名やファイル名に日本語を使うと、予期せぬエラーの原因になることがあるため、**半角英数字とハイフン（-）またはアンダースコア（_）**で名前をつけるのが一般的です。

（ここにスクリーンショットを挿入: デスクトップに `hello-world` フォルダを作成している様子）

### ⚙️ 2. VSCodeでフォルダを開く

次に、作成した `hello-world` フォルダを、VSCodeで開きます。

1.  VSCodeを起動します。
2.  左上のメニューから「ファイル」>「フォルダを開く...」を選択します。
3.  先ほど作成した `hello-world` フォルダを選択し、「開く」ボタンをクリックします。

（ここにスクリーンショットを挿入: VSCodeのメニューから「フォルダを開く」を選択し、`hello-world` フォルダを指定している様子）

フォルダを開くと、VSCodeの左側のサイドバー（エクスプローラー）に、`HELLO-WORLD` という名前が表示されます。これで、VSCodeがこのフォルダを「作業場所」として認識しました。

### ⚙️ 3. HTMLファイルの作成

いよいよ、最初のファイルを作成します。Webページの骨格を作るための言語、**HTML**で書かれたファイルです。

1.  VSCodeのエクスプローラーに表示されている `HELLO-WORLD` の名前にカーソルを合わせると、いくつかのアイコンが表示されます。一番左の、書類にプラスマークがついた「新しいファイル」アイコンをクリックします。
2.  ファイル名の入力欄が表示されるので、`index.html` と入力し、Enterキーを押します。

（ここにスクリーンショットを挿入: エクスプローラーで「新しいファイル」をクリックし、`index.html` と入力している様子）

`index.html` というファイルが作成され、右側のエディタ領域にそのファイルが開かれます。なぜ `index.html` という名前なのかは、また後のチュートリアルで詳しく説明しますが、今は「Webサイトのトップページに使われる、お決まりの名前」だと覚えておいてください。

### ⚙️ 4. HTMLコードの記述

作成した `index.html` に、以下のコードを**手で打ち込んで**みましょう。（コピー＆ペーストではなく、ぜひ自分でタイピングしてみてください。一文字ずつ打ち込むことが、上達への近道です）

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Page</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
</body>
</html>
```

VSCodeのコード補完機能が、あなたの入力を助けてくれるはずです。例えば、`<h1>` と打ち込むと、自動的に閉じタグの `</h1>` が補完されるのを体験してみてください。

**コードの簡単な説明:**

*   `<h1>Hello, World!</h1>`: `h1` は「一番大きな見出し（Heading 1）」を意味します。`Hello, World!` という文字を、一番大きな見出しとして表示する、という指示です。
*   `<p>This is my first web page.</p>`: `p` は「段落（Paragraph）」を意味します。文章のまとまりを表示するためのタグです。

今は、コードの意味を完全に理解する必要はありません。「こういう風に書くんだな」と感じるだけで十分です。

### ⚙️ 5. ブラウザでの表示確認

コードが書けたら、いよいよブラウザで表示してみましょう。

1.  作成した `index.html` ファイルを、エクスプローラー（Windows）またはFinder（Mac）で開きます。
2.  `index.html` ファイルのアイコンを、Google Chromeのアイコンの上にドラッグ＆ドロップするか、ファイルを右クリックして「プログラムから開く」>「Google Chrome」を選択します。

（ここにスクリーンショットを挿入: `index.html` ファイルをChromeで開いている様子）

すると、Chromeのウィンドウが開き、あなたが書いたコードが、Webページとして表示されます！

（ここにスクリーンショットを挿入: 「Hello, World!」と表示されたChromeの画面）

*   `<h1>` タグで囲んだ「Hello, World!」が、大きな文字の見出しとして表示されています。
*   `<p>` タグで囲んだ「This is my first web page.」が、本文として表示されています。

おめでとうございます！これが、あなたのプログラマーとしての、記念すべき第一歩です。

### ⚙️ 6. 修正とリロード

Web開発の面白さは、修正した結果がすぐに確認できる点にあります。

1.  VSCodeに戻り、`<h1>` の中の `Hello, World!` を、`こんにちは、世界！` に書き換えてみましょう。
2.  ファイルを保存します。（`Ctrl + S` or `Cmd + S`）
3.  Chromeのウィンドウに戻り、左上にある「リロード（再読み込み）」ボタン（円形の矢印アイコン）をクリックするか、`F5`キー（Macは `Cmd + R`）を押します。

すると、ブラウザの表示が、瞬時に「こんにちは、世界！」に切り替わるはずです。

この「**コードを修正 → 保存 → ブラウザをリロード**」という一連の流れは、Web開発の基本サイクルです。このサイクルを高速に回すことが、開発効率を上げる鍵となります。

## ✨ まとめ

このセクションでは、準備した開発環境が正常に動作するかを確認するために、簡単なWebページを作成し、表示するまでの一連の流れを体験しました。

*   **フォルダの作成**: 作業場所となるプロジェクトフォルダ（`hello-world`）を用意した。
*   **ファイルの作成**: VSCodeを使って、Webページの骨格となる `index.html` ファイルを作成した。
*   **コーディング**: 簡単なHTMLコードを記述し、「見出し」と「段落」を表示する指示を書いた。
*   **ブラウザ確認**: 作成したファイルをChromeで開き、意図通りに表示されることを確認した。
*   **修正とリロード**: コードを修正し、ブラウザをリロードすることで、変更が即座に反映されることを体験した。
