## 📌 BEエンジニア志望者へ

このChapterで学ぶレスポンシブデザインは、BEエンジニアとして**自分で実装できるようになる必要はありません**。ただし、以下の理由から、概念として理解しておくことが重要です。

## 詳細解説

### 📱 レスポンシブデザインとは？

**レスポンシブデザイン (Responsive Design)** とは、PC、タブレット、スマートフォンなど、**画面サイズの異なる様々なデバイスに対応**したWebデザインの手法です。

一つのHTMLファイルで、画面サイズに応じて、レイアウトや文字サイズなどを自動的に調整します。

```
【PC表示】                    【スマホ表示】
┌─────────────────────┐      ┌─────────┐
│  ヘッダー            │      │ ヘッダー │
├─────────┬───────────┤      ├─────────┤
│         │           │      │         │
│ メイン   │ サイド    │  →   │ メイン   │
│         │           │      │         │
├─────────┴───────────┤      ├─────────┤
│  フッター            │      │ サイド   │
└─────────────────────┘      ├─────────┤
                             │ フッター │
                             └─────────┘
```

### 🔧 実現方法：メディアクエリ

レスポンシブデザインは、CSSの**メディアクエリ (Media Query)** という機能で実現します。

```css
/* 基本のスタイル（全ての画面サイズに適用） */
.container {
  width: 100%;
}

/* 画面幅が768px以下の場合のみ適用 */
@media (max-width: 768px) {
  .sidebar {
    display: none;  /* サイドバーを非表示に */
  }
}
```

`@media (条件) { ... }` の中に書いたCSSは、条件を満たす場合にのみ適用されます。これにより、「PCでは横並び、スマホでは縦並び」といった切り替えが可能になります。

---

## ✨ まとめ

*   **レスポンシブデザイン**: 画面サイズに応じて、レイアウトが自動的に変わるデザイン手法
*   **メディアクエリ**: `@media (条件) { }` を使って、特定の画面サイズにのみCSSを適用する仕組み
*   **目的**: PC、タブレット、スマートフォンなど、様々なデバイスで快適に閲覧できるようにする

CSSのコードで `@media` を見かけたら、「画面サイズによって表示を切り替えている」と理解すれば大丈夫です。
