# Tutorial 5-1-5: リンクと画像の埋め込み

## 🎯 このセクションで学ぶこと

*   `<a>` タグを使って、他のWebページへのハイパーリンクを作成する方法を学ぶ。
*   `<img>` タグを使って、Webページに画像を埋め込む方法を学ぶ。
*   `href` 属性と `src` 属性という、重要な属性の役割を理解する。

---

## 導入

これまでのセクションで、HTMLを使って、テキストベースのコンテンツを構造化する方法を学びました。しかし、現代のWebページは、テキストだけでできているわけではありません。他のページへの**リンク**を辿ったり、カラフルな**画像**を楽しんだりすることで、Webは、より豊かで、便利なものになっています。

このセクションでは、WebをWebたらしめている、最も重要な2つの要素、「**リンク**」と「**画像**」を、あなたのWebページに追加する方法を学びます。

これを、「**旅行ガイドブックの作成**」に例えるなら、リンクは、「詳しくはP.35を参照」といった、関連ページへの案内書きです。そして、画像は、読者の興味を引く、美しい風景写真です。この2つが加わることで、あなたのガイドブック（Webページ）は、格段に魅力的で、使いやすいものになるでしょう。

---

## 詳細解説

### 🔗 ハイパーリンクの作成: `<a>` タグ

`<a>` タグは、**アンカー (Anchor)** の略で、ある場所から別の場所への「錨（いかり）」を降ろす、という意味合いを持っています。このタグを使うことで、テキストや画像をクリックしたときに、別のページにジャンプする、ハイパーリンクを作成できます。

`<a>` タグの最も重要な特徴は、**`href` 属性** を持つことです。`href` は、**Hypertext Reference** の略で、リンク先の「住所（URL）」を指定します。

```html
<a href="https://www.google.com">Googleへのリンク</a>
```

*   **`<a>`**: 「ここからここまでがリンクですよ」と定義します。
*   **`href="..."`**: リンクの**飛び先**を指定する、最も重要な属性です。ここに、目的のWebページのURLを記述します。
*   **`Googleへのリンク`**: 実際に、ブラウザの画面に表示される、クリック可能なテキストです。この部分を、**アンカーテキスト**と呼びます。

#### ページ内リンク

`<a>` タグは、外部のWebページだけでなく、同じページ内の、特定の場所にジャンプするためにも使えます。これを、ページ内リンク（またはアンカージャンプ）と呼びます。

1.  まず、ジャンプ先の目印となる要素に、`id` 属性を使って、ユニークな名前を付けます。

    ```html
    <h2 id="section-2">第2章: CSSの冒険</h2>
    ```

2.  次に、リンク元の `<a>` タグの `href` 属性に、`#`（ハッシュ）に続けて、その `id` 名を指定します。

    ```html
    <a href="#section-2">第2章へジャンプ</a>
    ```

これにより、「第2章へジャンプ」というリンクをクリックすると、同じページ内の「第2章: CSSの冒険」という見出しまで、スムーズにスクロールします。長いWebページで、目次を作成する際などに、非常に便利な機能です。

### 🖼️ 画像の埋め込み: `<img>` タグ

`<img>` タグは、**イメージ (Image)** の略で、Webページに画像を埋め込むために使います。

`<img>` タグの最も重要な特徴は、**`src` 属性** を持つことです。`src` は、**Source** の略で、表示したい画像ファイルが、どこにあるのか、その「場所（パス）」を指定します。

また、`<img>` タグは、`<h1>` や `<p>` とは異なり、**終了タグ（`</img>`）を持ちません**。このように、単独で完結するタグを、**空要素 (Empty Element)** と呼びます。

```html
<img src="images/cat.jpg" alt="かわいい猫の画像">
```

*   **`<img>`**: 画像を埋め込むことを宣言します。
*   **`src="..."`**: 表示したい**画像ファイルの場所**を指定する、最も重要な属性です。Web上のURL（例: `https://.../image.png`）や、自分のコンピュータ上のファイルパス（例: `images/cat.jpg`）を指定します。
*   **`alt="..."`**: **代替テキスト (Alternative Text)** を指定する、**非常に重要な属性**です。もし、何らかの理由で画像が表示されなかった場合（パスが間違っている、サーバーがダウンしているなど）、このテキストが代わりに表示されます。また、目の不自由な方が使う、スクリーンリーダー（音声読み上げソフト）は、この `alt` テキストを読み上げることで、画像の内容を伝えます。全ての `<img>` タグには、必ず、その画像の内容を的確に説明する `alt` テキストを記述するように、心がけましょう。

#### 画像のサイズ指定

`width`（幅）と `height`（高さ）属性を使って、画像の表示サイズをピクセル単位で指定することもできます。

```html
<img src="images/dog.jpg" alt="元気な犬の画像" width="300" height="200">
```

ただし、現在では、画像のサイズ調整は、CSSで行うのが一般的です。HTMLでは、`alt` テキストの指定に集中し、見た目の調整はCSSに任せる、という役割分担が推奨されています。

## ✨ まとめ

このセクションでは、Webページを、よりインタラクティブで、視覚的に豊かにするための、リンクと画像の埋め込み方法を学びました。

| タグ | 属性 | 役割 |
| :--- | :--- | :--- |
| `<a>` | `href` | 他のページや、ページ内の特定箇所への**ハイパーリンク**を作成する。`href` に、リンク先のURLやIDを指定する。 |
| `<img>` | `src`, `alt` | **画像**を埋め込む。`src` に、画像ファイルの場所を指定し、`alt` には、必ず、画像の代替テキストを記述する。 |

*   **`href` (Hypertext Reference)**: リンクの**飛び先**を指定する。
*   **`src` (Source)**: 画像の**源（みなもと）**を指定する。

この2つのタグを使いこなすことで、あなたのWebページは、単なる静的な文書から、他の情報と繋がり、視覚的な魅力を持つ、ダイナミックなメディアへと進化します。

これで、HTMLの基礎の基礎は、一通り学び終えました。次のチャプターでは、Webページの「見た目」を司る、CSSの世界に足を踏み入れます。
