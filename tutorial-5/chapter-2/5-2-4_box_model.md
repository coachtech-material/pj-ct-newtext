# Tutorial 5-2-4: ボックスモデル

## 🎯 このセクションで学ぶこと

*   全てのHTML要素が、「箱（ボックス）」として扱われるという、CSSの基本的な概念を理解する。
*   ボックスを構成する4つの領域（Content, Padding, Border, Margin）の役割を理解する。

---

## 詳細解説

Webページ上の、見出し、段落、画像など、全てのHTML要素は、実は、目に見えない「**箱（ボックス）**」に入っている、と考えることができます。この概念を、**ボックスモデル (Box Model)** と呼びます。

### 📦 ボックスの4つの領域

CSSのボックスモデルでは、一つの「箱」は、以下の4つの領域から構成されています。

```
┌─────────────────────────────────────┐
│             Margin（外側の余白）      │
│   ┌─────────────────────────────┐   │
│   │         Border（枠線）        │   │
│   │   ┌─────────────────────┐   │   │
│   │   │   Padding（内側の余白） │   │   │
│   │   │   ┌─────────────┐   │   │   │
│   │   │   │   Content   │   │   │   │
│   │   │   │  （内容）    │   │   │   │
│   │   │   └─────────────┘   │   │   │
│   │   └─────────────────────┘   │   │
│   └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

| 領域 | 役割 | 例え |
| :--- | :--- | :--- |
| **Content** | テキストや画像など、要素の内容が表示される中心の領域 | 箱の中身 |
| **Padding** | コンテンツと枠線の間の、**内側の余白** | 箱の中の緩衝材 |
| **Border** | パディングの外側にある、**枠線** | 箱の外枠 |
| **Margin** | ボックスの最も外側にある、**他の要素との間の余白** | 箱と箱の間隔 |

### 📝 よく使うプロパティ

```css
div {
  /* コンテンツ領域のサイズ */
  width: 300px;
  height: 200px;

  /* 内側の余白（上下左右すべて20px） */
  padding: 20px;

  /* 枠線（2pxの黒い実線） */
  border: 2px solid #000;

  /* 外側の余白（上下左右すべて30px） */
  margin: 30px;
}
```

---

## ✨ まとめ

*   全てのHTML要素は、**Content**, **Padding**, **Border**, **Margin** の4つの領域から成る「箱」として扱われる。
*   **Padding**: 箱の**内側**の余白
*   **Border**: 箱の**枠線**
*   **Margin**: 箱の**外側**の余白（他の要素との距離）

CSSのコードで `padding` や `margin` を見かけたら、「内側の余白」「外側の余白」と読み替えると、理解しやすくなります。
