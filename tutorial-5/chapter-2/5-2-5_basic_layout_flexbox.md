# Tutorial 5-2-5: レイアウトの基礎（Flexbox）

## 🎯 このセクションで学ぶこと

*   モダンCSSレイアウトの主流である、Flexboxの基本的な概念を理解する。
*   FlexコンテナとFlexアイテムの親子関係を理解する。
*   `display: flex` を使って、要素を横並びにする方法を学ぶ。
*   `justify-content` と `align-items` を使って、アイテムの配置を自在にコントロールする方法を学ぶ。

---

## 導入

これまでのセクションで、一つ一つの「箱（ボックス）」のサイズや余白を調整する方法を学びました。このセクションでは、いよいよ、それらの「箱」を、ページ上に、**どのように並べるか**、という、レイアウトの技術について学びます。

かつて、CSSで、要素を横並びに配置するのは、非常にトリッキーで、難しい作業でした。しかし、**Flexbox (Flexible Box Layout Module)** の登場によって、それが、驚くほど簡単で、直感的に行えるようになりました。

Flexboxは、その名の通り、アイテムのサイズや順序、配置を、**柔軟に（flexibly）**コントロールするための、強力なレイアウトモデルです。これをマスターすれば、ヘッダーのナビゲーションメニューや、カード型のコンテンツリストなど、現代的なWebサイトで頻繁に見られる、多くのレイアウトパターンを、簡単に実装できます。

ここでは、Flexboxを、「**本棚の整理**」に例えて、考えてみましょう。本棚（親要素）に、いくつかの本（子要素）を並べる際に、「横一列に並べる」「中央揃えにする」「均等な間隔で配置する」といった指示を、簡単に出せるのが、Flexboxのすごいところです。

---

## 詳細解説

Flexboxを使い始めるには、たった一つのことから始めます。それは、**親要素**に対して、`display: flex;` を指定することです。

### 👨‍👩‍👧‍👦 FlexコンテナとFlexアイテム

Flexboxのレイアウトは、**親要素（Flexコンテナ）**と、その直下にある**子要素（Flexアイテム）**という、明確な親子関係で成り立っています。

```html
<div class="container"> <!-- これがFlexコンテナ -->
  <div class="item">アイテム1</div> <!-- これがFlexアイテム -->
  <div class="item">アイテム2</div>
  <div class="item">アイテム3</div>
</div>
```

```css
.container {
  display: flex;
}
```

親要素である `.container` に `display: flex;` を指定した瞬間に、魔法が起こります。通常、`<div>` タグは、縦に並ぶブロックレベル要素ですが、Flexコンテナの中の子要素（Flexアイテム）は、自動的に、**横一列に並びます**。

*   **Flexコンテナ**: `display: flex;` が指定された親要素。アイテムを整列させるための、様々なプロパティを持つ。
*   **Flexアイテム**: Flexコンテナの直接の子要素。コンテナのルールに従って、配置される。

### ↔️ アイテムの水平方向の配置: `justify-content`

`justify-content` は、Flexコンテナの**主軸（デフォルトでは水平方向）**に沿って、Flexアイテムを、どのように配置するかを決定します。これは、最もよく使うプロパティの一つです。

```css
.container {
  display: flex;
  justify-content: ...; /* ここに以下の値が入る */
}
```

*   `flex-start` (デフォルト): コンテナの**左端**に詰めて配置。
*   `flex-end`: コンテナの**右端**に詰めて配置。
*   `center`: コンテナの**中央**に配置。
*   `space-between`: **両端**にアイテムを配置し、残りのアイテムは、その間で**均等な間隔**を空けて配置。
*   `space-around`: 全てのアイテムの**周り**に、均等な間隔を空けて配置。（両端のアイテムの外側にも、少し間隔ができます）
*   `space-evenly`: 全てのアイテムの**間**と、**両端**に、完全に均等な間隔を空けて配置。

### ↕️ アイテムの垂直方向の配置: `align-items`

`align-items` は、Flexコンテナの**交差軸（デフォルトでは垂直方向）**に沿って、Flexアイテムを、どのように配置するかを決定します。

```css
.container {
  display: flex;
  height: 200px; /* コンテナに高さがないと、効果がわからないため */
  align-items: ...; /* ここに以下の値が入る */
}
```

*   `stretch` (デフォルト): アイテムの高さを、コンテナの高さに合わせて、**引き伸ばす**。
*   `flex-start`: コンテナの**上端**に詰めて配置。
*   `flex-end`: コンテナの**下端**に詰めて配置。
*   `center`: コンテナの**中央（垂直方向）**に配置。
*   `baseline`: アイテム内のテキストの、ベースライン（下線）を揃える。

![align-itemsの図解](https://.../align-items.png)  
*ここに、各値の効果を視覚的に示す図を挿入します。*

`justify-content: center;` と `align-items: center;` を組み合わせることで、アイテムを、コンテナの**ど真ん中（水平・垂直中央）**に配置する、という、かつては非常に難しかったレイアウトが、いとも簡単に実現できます。

```css
.container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 300px;
}
```

## ✨ まとめ

このセクションでは、モダンCSSレイアウトの強力なツールである、Flexboxの、最も基本的な使い方を学びました。

*   **Flexboxの起動**: 親要素に `display: flex;` を指定するだけで、子要素が横並びになる。
*   **親子関係**: Flexboxは、**Flexコンテナ（親）**と**Flexアイテム（子）**で構成される。
*   **水平方向の配置**: `justify-content` プロパティで、アイテムの横方向の揃え方を指定する。（`center`, `space-between` など）
*   **垂直方向の配置**: `align-items` プロパティで、アイテムの縦方向の揃え方を指定する。（`center` など）

Flexboxには、ここで紹介した以外にも、アイテムの折り返し (`flex-wrap`)、順序の変更 (`order`)、個々のアイテムの成長・縮小率 (`flex-grow`, `flex-shrink`) など、多くの強力な機能があります。しかし、まずは、この `display: flex`, `justify-content`, `align-items` の3つをマスターするだけで、作れるレイアウトの幅が、劇的に広がることを、実感できるはずです。
