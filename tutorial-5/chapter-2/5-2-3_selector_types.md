# Tutorial 5-2-3: セレクタの種類

## 🎯 このセクションで学ぶこと

*   特定のHTML要素を、正確に「狙い撃ち」するための、様々なセレクタの種類を学ぶ。
*   タグセレクタ、クラスセレクタ、IDセレクタという、最も基本的な3つのセレクタを使いこなせるようになる。
*   セレクタを組み合わせることで、より複雑な指定ができることを理解する。

---

## 導入

前のセクションで、CSSが `セレクタ { プロパティ: 値; }` という構文で書かれることを学びました。この「セレクタ」は、「**どの要素のスタイルを変更するのか**」を指定する、非常に重要な部分です。

これまでは、`h1` や `p` といった、タグ名をそのまま使う「タグセレクタ」だけを見てきました。しかし、実際のWebページでは、「全ての段落ではなく、この特定の段落だけ、文字を赤くしたい」とか、「リストの中でも、重要な項目だけ、背景色を変えたい」といった、より細かい指定が必要になります。

このセクションでは、HTML要素を、より自由に、より正確に選択するための、様々なセレクタの種類を学びます。セレクタを使いこなすことは、CSSをマスターするための鍵です。これを、「**的当てゲーム**」に例えるなら、様々な種類の「矢」を手に入れるようなものです。小さな的を狙うための精密な矢、広範囲を一度に狙うための散弾の矢など、状況に応じて、最適な矢（セレクタ）を選ぶ技術を、身につけていきましょう。

---

## 詳細解説

### 🎯 基本的な3つのセレクタ

まずは、最も頻繁に使われる、3つの基本的なセレクタをマスターしましょう。

#### 1. タグセレクタ (Tag Selector)

*   **書き方**: HTMLタグの名前を、そのまま記述します。
*   **役割**: 指定された**全てのタグ**に、同じスタイルを適用します。
*   **使い所**: サイト全体の、基本的なスタイル（ベーススタイル）を設定するのに適しています。例えば、「全ての段落の文字サイズを16pxにする」といった、全体的なルールに使います。

```css
/* 全てのpタグの文字色を灰色にする */
p {
  color: gray;
}

/* 全てのh2タグの下に、線を引く */
h2 {
  border-bottom: 1px solid #ccc;
}
```

#### 2. クラスセレクタ (Class Selector)

*   **書き方**: ピリオド `.` の後に、任意のクラス名を記述します。（例: `.class-name`）
*   **役割**: HTML要素に `class` 属性を使って名前を付け、その**クラス名を持つ全ての要素**に、スタイルを適用します。
*   **使い所**: スタイルを**使い回したい**場合に、非常に便利です。「警告文」用の赤い文字スタイルや、「ボタン」用のデザインなど、サイト内の様々な場所で、繰り返し使われるデザイン要素に対して使います。
*   **特徴**: 一つの要素に、複数のクラス名を（スペースで区切って）指定したり、同じクラス名を、ページ内の複数の要素に、何度でも使うことができます。

**HTML側:**
```html
<p class="warning">注意: この操作は取り消せません。</p>
<div class="warning">システムエラーが発生しました。</div>

<button class="btn btn-primary">送信</button>
<button class="btn">キャンセル</button>
```

**CSS側:**
```css
/* "warning" というクラスを持つ全ての要素の文字色を赤にする */
.warning {
  color: red;
  font-weight: bold;
}

/* "btn" というクラスを持つ要素の基本スタイル */
.btn {
  padding: 10px 20px;
  border: 1px solid #ccc;
}

/* "btn-primary" というクラスも併せ持つ要素のスタイル */
.btn-primary {
  background-color: blue;
  color: white;
}
```

#### 3. IDセレクタ (ID Selector)

*   **書き方**: ハッシュ `#` の後に、任意のID名を記述します。（例: `#id-name`）
*   **役割**: HTML要素に `id` 属性を使って名前を付け、その**ID名を持つ、たった一つの要素**に、スタイルを適用します。
*   **使い所**: ページ内で、**絶対に一つしか存在しない、ユニークな要素**に対して使います。例えば、サイトのロゴ、メインナビゲーション、フッターなど、ページの主要な構成要素に対して使います。
*   **注意点**: `id` 属性の値は、**一つのHTMLページ内で、絶対に重複してはいけません**。クラスが「グループ名」だとしたら、IDは「個人名（マイナンバー）」のようなものです。

**HTML側:**
```html
<header id="main-header">
  <h1>サイトのタイトル</h1>
</header>
```

**CSS側:**
```css
/* "main-header" というIDを持つ要素に、背景色を設定する */
#main-header {
  background-color: #333;
  color: white;
  padding: 20px;
}
```

### 組み合わせと階層

セレクタは、組み合わせることで、より複雑な指定が可能になります。

#### 子孫セレクタ

*   **書き方**: セレクタを、スペースで区切って、複数並べます。（例: `親要素 子要素`）
*   **役割**: 「親要素の中にある、全ての子孫要素」を選択します。

```css
/* "#main-contents" というIDを持つ要素の中にある、全てのpタグ */
#main-contents p {
  line-height: 1.8;
}

/* "article" タグの中にある、"warning" クラスを持つ要素 */
article .warning {
  border: 1px solid red;
}
```

これは、特定の範囲内だけで、スタイルを限定したい場合に、非常に便利です。

## ✨ まとめ

このセクションでは、CSSの心臓部である「セレクタ」の、基本的な種類について学びました。

| セレクタの種類 | 書き方 | 選択対象 | 特徴 |
| :--- | :--- | :--- | :--- |
| タグセレクタ | `p` | 指定された全てのタグ | 全体的なベーススタイルに使う |
| クラスセレクタ | `.classname` | 指定されたクラスを持つ全ての要素 | スタイルの使い回しに最適 |
| IDセレクタ | `#idname` | 指定されたIDを持つ唯一の要素 | ページに一つだけの要素に使う |
| 子孫セレクタ | `div p` | `div`の中にある全ての`p` | 特定の範囲にスタイルを限定する |

**クラスとIDの使い分けの基本方針:**

*   **クラス (Class)**: 繰り返し使う可能性がある、汎用的なスタイルに使う。（例: `.btn`, `.warning`, `.card`）
*   **ID (ID)**: そのページで、たった一度しか使わない、特定の要素に使う。（例: `#header`, `#footer`, `#main-visual`）

これらのセレクタを、適切に使い分けることが、整理されていて、メンテナンスしやすいCSSを書くための、第一歩です。
