---

## 詳細解説

### 📜 CSSの基本構文

CSSのルールは、以下の3つの要素で構成されます。

```css
セレクタ {
  プロパティ: 値;
}
```

*   **セレクタ (Selector)**: 「**誰の**」スタイルを変更したいのかを指定します。HTMLタグ名（例: `h1`, `p`）などを指定します。
*   **プロパティ (Property)**: 「**何を**」変更したいのかを指定します。`color`（文字色）や、`font-size`（文字の大きさ）などです。
*   **値 (Value)**: 「**どうする**」のか、具体的な値を指定します。`blue`（青色）や、`16px`（16ピクセル）などです。

```css
/* h1タグの、文字色を、青色にする */
h1 {
  color: blue;
}

/* pタグの、文字の大きさを16ピクセルにする */
p {
  font-size: 16px;
}
```

### 🔌 HTMLにCSSを適用する方法

書いたCSSを、HTMLに適用するには、主に以下の方法があります。

#### 外部スタイルシート (External Style Sheet) - 推奨

CSSのコードを、`.css` という拡張子を持つ、独立したファイル（例: `style.css`）として保存します。そして、HTMLファイルの `<head>` タグの中から、`<link>` タグを使って、そのCSSファイルを読み込みます。

**1. CSSファイルの作成 (`style.css`)**

```css
/* style.css */
h1 {
  color: purple;
}
```

**2. HTMLファイルからの読み込み (`index.html`)**

```html
<!DOCTYPE html>
<html>
<head>
  <title>外部スタイルシート</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>この見出しは紫色になります</h1>
</body>
</html>
```

`<link>` タグの属性:
*   `rel="stylesheet"`: 読み込むファイルが、「スタイルシート」であることを示します。
*   `href="style.css"`: 読み込むCSSファイルの場所（パス）を指定します。

> 📝 **なぜ外部スタイルシートが推奨されるのか？**
> 
> HTMLから、完全に見た目を分離できるため、メンテナンス性が最も高くなります。一つのCSSファイルを、サイト内の全てのHTMLファイルで共有すれば、サイト全体のデザインを、一元管理できます。

---

## ✨ まとめ

*   **CSSの基本構文**: `セレクタ { プロパティ: 値; }` という形式で、「**誰の、何を、どうする**」を指定する。
*   **外部スタイルシート**: CSSを別ファイルに分離し、`<link>` タグで読み込む方法。メンテナンス性が高く、最も推奨される。
