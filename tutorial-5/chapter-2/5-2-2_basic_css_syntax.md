# Tutorial 5-2-2: CSSの基本的な書き方

## 🎯 このセクションで学ぶこと

*   CSSの基本的な書式である、「セレクタ」「プロパティ」「値」の3つの要素を理解する。
*   HTMLファイルにCSSを適用するための、3つの方法（インライン、内部、外部）を学ぶ。
*   3つの方法の中で、なぜ「外部スタイルシート」が最も推奨されるのかを理解する。

---

## 導入

前のセクションで、CSSがWebページの見た目を担当する言語であることを学びました。では、具体的に、どのように書けば、HTML要素のスタイルを変更できるのでしょうか？

CSSの文法は、非常にシンプルで、直感的です。それは、「**誰の（セレクタ）、何を（プロパティ）、どうする（値）**」という、たった3つの要素で構成されています。

このセクションでは、まず、このCSSの基本的な「書き方のルール」を学びます。その後、書いたCSSを、どのようにしてHTMLファイルに読み込ませるのか、その3つの方法について、詳しく見ていきましょう。

---

## 詳細解説

### 📜 CSSの基本構文

CSSのルールは、以下の3つの要素から成る、一つの「ルールセット」で構成されます。

```css
セレクタ {
  プロパティ: 値;
}
```

*   **セレクタ (Selector)**: 「**誰の**」スタイルを変更したいのかを指定します。HTMLタグ名（例: `h1`, `p`）や、特定の`id`、`class`などを指定します。（セレクタの詳細は、次のセクションで学びます）
*   **プロパティ (Property)**: 「**何を**」変更したいのかを指定します。`color`（文字色）や、`font-size`（文字の大きさ）、`background-color`（背景色）など、変更したいスタイルの種類を指定します。
*   **値 (Value)**: 「**どうする**」のか、具体的な値を指定します。`blue`（青色）や、`16px`（16ピクセル）、`#ffffff`（白色）など、プロパティに対する、具体的な設定値を指定します。

**ルールセット**は、波括弧 `{}` で囲み、その中に、一つ以上の**宣言**（`プロパティ: 値;` のペア）を記述します。各宣言は、必ずセミコロン `;` で終わります。

```css
/* h1タグの、文字色を、青色にする */
h1 {
  color: blue;
}

/* pタグの、文字の大きさを16ピクセルにし、背景色を薄い灰色にする */
p {
  font-size: 16px;
  background-color: #f0f0f0;
}
```

### 🔌 HTMLにCSSを適用する3つの方法

書いたCSSを、HTMLに適用するには、大きく分けて3つの方法があります。

#### 方法1: インラインスタイル (Inline Styles)

*   **書き方**: スタイルを適用したいHTMLタグの中に、直接 `style` 属性を使って、CSSを記述します。
*   **特徴**: 特定の一つの要素にだけ、固有のスタイルを適用したい場合に、手軽に使えます。

```html
<h1 style="color: red; font-size: 24px;">この見出しだけ赤色になります</h1>
<p>この段落は、影響を受けません。</p>
```

*   **デメリット**: HTMLとCSSが混在するため、「構造と見た目の分離」の原則に反します。スタイルの再利用ができず、修正も大変なため、**基本的には、この方法は避けるべき**です。テストや、どうしても他で代替できない場合に、限定的に使用します。

#### 方法2: 内部スタイルシート (Internal Style Sheet)

*   **書き方**: HTMLファイルの `<head>` タグの中に、`<style>` タグを設置し、その中にCSSを記述します。
*   **特徴**: そのHTMLファイル**一枚にだけ**適用されるスタイルを、一箇所にまとめて記述できます。

```html
<!DOCTYPE html>
<html>
<head>
  <title>内部スタイルシート</title>
  <style>
    h1 {
      color: green;
    }
    p {
      font-size: 14px;
    }
  </style>
</head>
<body>
  <h1>このページの見出しは緑色になります</h1>
  <p>このページの段落は14ピクセルになります。</p>
</body>
</html>
```

*   **デメリット**: スタイルを、他のHTMLファイルで再利用することができません。サイト全体で、共通のデザインを使いたい場合には、不向きです。

#### 方法3: 外部スタイルシート (External Style Sheet)

*   **書き方**: CSSのコードを、`.css` という拡張子を持つ、完全に独立したファイル（例: `style.css`）として保存します。そして、HTMLファイルの `<head>` タグの中から、`<link>` タグを使って、そのCSSファイルを読み込みます。
*   **特徴**: これが、**最も推奨される、標準的な方法**です。

**1. CSSファイルの作成 (`style.css`)**

```css
/* style.css */
h1 {
  color: purple;
}
```

**2. HTMLファイルからの読み込み (`index.html`)**

```html
<!DOCTYPE html>
<html>
<head>
  <title>外部スタイルシート</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <h1>この見出しは紫色になります</h1>
</body>
</html>
```

`<link>` タグの属性:
*   `rel="stylesheet"`: 読み込むファイルが、「スタイルシート」であることを示します。
*   `href="style.css"`: 読み込むCSSファイルの場所（パス）を指定します。

*   **メリット**: HTMLから、完全に見た目を分離できるため、メンテナンス性、再利用性が、最も高くなります。一つのCSSファイルを、サイト内の全てのHTMLファイルで共有すれば、サイト全体のデザインを、一元管理できます。

## ✨ まとめ

このセクションでは、CSSの基本的な書き方と、HTMLへの適用方法について学びました。

*   **CSSの基本構文**: `セレクタ { プロパティ: 値; }` という形式で、「**誰の、何を、どうする**」を定義する。
*   **HTMLへの適用方法**: 3つの方法があるが、**外部スタイルシート**が、最も推奨される。

| 方法 | 書き場所 | 影響範囲 | 推奨度 |
| :--- | :--- | :--- | :--- |
| インライン | HTMLタグの`style`属性 | そのタグのみ | 低 (避けるべき) |
| 内部 | HTMLの`<head>`内の`<style>`タグ | そのHTMLファイルのみ | 中 (限定的に使用) |
| 外部 | 別の`.css`ファイル | 読み込んだ全てのHTML | **高 (常にこれを目指す)** |

常に「外部スタイルシート」を使うことを基本とし、HTMLファイルとCSSファイルを、きちんと分けて管理する癖をつけましょう。これが、美しく、メンテナンスしやすいWebサイトを作るための、プロフェッショナルなアプローチです。
