# Tutorial 5-2-6: 色とフォントの指定

## 🎯 このセクションで学ぶこと

*   CSSで色を指定するための、様々な方法（キーワード、HEX、RGB）を理解する。
*   文字の色 (`color`) と、背景色 (`background-color`) を設定する方法を学ぶ。
*   Webページで使われるフォントを指定する (`font-family`) 方法と、その注意点を学ぶ。
*   文字の大きさ (`font-size`) や、太さ (`font-weight`) を調整する方法を学ぶ。

---

## 導入

レイアウトがWebページの「骨格」だとしたら、**色**と**フォント**は、そのページの「表情」や「個性」を決定づける、非常に重要なデザイン要素です。適切な配色と、読みやすいフォントを選ぶだけで、Webサイトの印象は、劇的に向上します。

このセクションでは、CSSを使って、Webページの「色」と「フォント」を、どのように指定するのかを学びます。まるで、「**パレットと絵筆**」を手に入れるように、これらのプロパティを使いこなすことで、あなたのWebページに、豊かな彩りと、洗練された雰囲気を与えることができます。

---

## 詳細解説

### 🎨 色の指定方法

CSSで色を指定するには、主に3つの方法があります。

#### 1. カラーキーワード (Color Keywords)

*   **書き方**: `red`, `blue`, `green`, `black`, `white` のように、あらかじめ定義された色の名前を、そのまま記述します。
*   **特徴**: `lightgray` や `skyblue` など、約140種類の色名が定義されており、手軽で直感的です。ただし、微妙な色合いを表現することはできません。

```css
p {
  color: darkslateblue; /* 文字色を濃いスレートブルーに */
}
```

#### 2. HEX値 (16進数カラーコード)

*   **書き方**: ハッシュ `#` の後に、赤(R)、緑(G)、青(B)の光の三原色の強さを、それぞれ2桁の16進数（00〜FF）で表現した、6桁のコードを記述します。（例: `#RRGGBB`）
*   **特徴**: Webデザインで、**最も一般的に使われる**色の指定方法です。約1677万色を、正確に表現することができます。デザインツール（Photoshop, Figmaなど）で色を選ぶと、通常、このHEX値で色が示されます。
*   `#FF0000` は、赤が最大、緑と青が0なので、純粋な赤色になります。
*   `#000000` は、全ての色が0なので、黒色になります。
*   `#FFFFFF` は、全ての色が最大なので、白色になります。
*   R, G, B の各値が、同じ2桁の繰り返しの場合（例: `#112233`）、`#123` のように、3桁に省略して書くこともできます。

```css
body {
  background-color: #f4f4f4; /* 背景色を、非常に薄い灰色に */
}

h2 {
  color: #333333; /* 見出しの文字色を、濃い灰色に */
}
```

#### 3. RGB値 / RGBA値

*   **書き方**: `rgb(R, G, B)` の形式で、赤、緑、青の強さを、それぞれ0〜255の数値で指定します。
*   **特徴**: HEX値と同じく、全ての色を表現できます。`rgba(R, G, B, A)` の形式を使うと、4つ目の値 `A` (アルファ値) で、**透明度**を指定できるのが、大きなメリットです。アルファ値は、0（完全に透明）から1（完全に不透明）の間の数値で指定します。

```css
.overlay {
  /* 黒色の、半透明の背景 */
  background-color: rgba(0, 0, 0, 0.5);
}
```

### 🖋️ フォント関連のプロパティ

#### 1. フォントファミリーの指定: `font-family`

*   **役割**: Webページで使われるフォントの種類を指定します。
*   **書き方**: 複数のフォント名を、カンマ `,` で区切って、優先順位の高い順に記述します。

```css
body {
  font-family: "Helvetica Neue", Arial, "Hiragino Kaku Gothic ProN", Meiryo, sans-serif;
}
```

*   **なぜ複数指定するのか？**: ユーザーが閲覧しているコンピュータに、最初に指定したフォント（この例では `Helvetica Neue`）がインストールされていない場合に備えて、代替のフォントを、順番に指定しておくためです。もし、指定されたフォントが、どれもインストールされていなかった場合、最後の「総称フォントファミリー」（この例では `sans-serif`）が使われます。
*   **総称フォントファミリー**: `serif`（明朝体のような、飾りのあるフォント）、`sans-serif`（ゴシック体のような、飾りのないフォント）、`monospace`（等幅フォント）など、具体的なフォント名ではなく、フォントの「種類」を指定します。これは、**必ず、リストの最後に記述**します。

#### 2. 文字の大きさ: `font-size`

*   **役割**: 文字の大きさを指定します。
*   **単位**: `px` (ピクセル), `em`, `rem`, `%` など、様々な単位がありますが、初学者のうちは、最も直感的な `px` を使えれば十分です。

```css
body {
  font-size: 16px; /* 基準となる文字サイズ */
}

h1 {
  font-size: 32px; /* 見出しは大きく */
}

.caption {
  font-size: 12px; /* 注釈は小さく */
}
```

#### 3. 文字の太さ: `font-weight`

*   **役割**: 文字の太さを指定します。
*   **書き方**: `normal` (通常), `bold` (太字) といったキーワードや、100〜900の数値で指定します。（`normal` は 400, `bold` は 700 に相当します）

```css
.important {
  font-weight: bold;
}
```

## ✨ まとめ

このセクションでは、Webページの「表情」を作る、色とフォントの指定方法について学びました。

*   **色の指定**: `red` のような**キーワード**、`#RRGGBB` 形式の **HEX値**、`rgba(R, G, B, A)` 形式の **RGBA値** などがある。特に、透明度を指定できるRGBAは便利。
*   **文字色**: `color` プロパティで指定する。
*   **背景色**: `background-color` プロパティで指定する。
*   **フォントの種類**: `font-family` プロパティで、複数のフォントを、優先順位を付けて指定する。
*   **文字の大きさ**: `font-size` プロパティで指定する。
*   **文字の太さ**: `font-weight` プロパティで指定する。

これらのプロパティを組み合わせることで、Webサイトのブランドイメージに合った、統一感のあるデザインを、作り出すことができます。
