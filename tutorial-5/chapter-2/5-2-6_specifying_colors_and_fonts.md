# Tutorial 5-2-6: 色とフォントの指定

## 🎯 このセクションで学ぶこと

*   CSSで色を指定するための方法を理解する。
*   文字の色や背景色、フォントサイズの指定方法を知る。

---

## 詳細解説

### 🎨 色の指定方法

CSSで色を指定するには、主に以下の方法があります。

#### 1. カラーキーワード

`red`, `blue`, `green`, `black`, `white` のように、色の名前をそのまま記述します。

```css
p {
  color: red;
}
```

#### 2. HEX値（16進数カラーコード）

`#` の後に、6桁の16進数で色を指定します。**最もよく使われる方法**です。

```css
h1 {
  color: #333333;        /* 濃い灰色 */
  background-color: #f4f4f4;  /* 薄い灰色 */
}
```

> 📝 **ポイント**: デザインツール（Figmaなど）で色を選ぶと、このHEX値で表示されることが多いです。`#000000` は黒、`#ffffff` は白です。

### 🖋️ フォント関連のプロパティ

| プロパティ | 役割 | 例 |
| :--- | :--- | :--- |
| `color` | 文字の色 | `color: #333;` |
| `background-color` | 背景色 | `background-color: #f4f4f4;` |
| `font-size` | 文字の大きさ | `font-size: 16px;` |
| `font-weight` | 文字の太さ | `font-weight: bold;` |

```css
body {
  color: #333;           /* 文字色を濃い灰色に */
  font-size: 16px;       /* 基本の文字サイズ */
}

h1 {
  font-size: 32px;       /* 見出しは大きく */
  font-weight: bold;     /* 太字に */
}
```

---

## ✨ まとめ

*   **色の指定**: カラーキーワード（`red`）やHEX値（`#333333`）で指定する
*   **文字色**: `color` プロパティ
*   **背景色**: `background-color` プロパティ
*   **文字サイズ**: `font-size` プロパティ
*   **文字の太さ**: `font-weight` プロパティ

CSSのコードを読むときは、これらのプロパティ名から「何を変更しているか」を推測できるようになりましょう。
