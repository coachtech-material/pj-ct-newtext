# Tutorial 1-1: プログラミングとは

## 🎯 このセクションで学ぶこと

*   プログラミングが「コンピュータへの指示書」であることを、身近な例えで理解する。
*   なぜ今、多くの人がプログラミングを学ぶのか、その重要性を知る。
*   プログラミングで何が作れるのか、具体的なイメージを持つことで、学習のゴールを明確にする。

---

## 導入

はじめまして！これから皆さんと一緒に、プログラミングの世界を探求していくことになりました。どうぞよろしくお願いします。

さて、「プログラミング」と聞くと、どんなイメージが湧きますか？

「なんだか難しそう…」「黒い画面に謎の英語を打ち込むやつでしょ？」「数学が得意な人がやるもの？」

もし、そんな風に感じていたとしても、全く心配いりません。この教材は、まさにそうした「プログラミング未経験」の方々のために作られています。専門用語はできるだけ日常の言葉に置き換え、一つ一つ丁寧に進めていくので、安心してくださいね。

まずは、一番大事な質問から始めましょう。**そもそも、プログラミングって何なのでしょうか？**

一番シンプルな答えは、「**コンピュータにやってほしいことを、順番に伝えるための指示書**」です。

### 比喩で理解する: プログラミングは「料理のレシピ」にそっくり！

想像してみてください。あなたは凄腕のシェフ（コンピュータ）に、最高のカレーライスを作ってもらいたいと思っています。しかし、このシェフは非常に優秀ですが、一つだけ困った特徴があります。それは、「**指示されたことしか、その通りにしかできない**」ということです。

あなたが「カレーを作って」とだけ伝えても、シェフは困ってしまいます。

*   「どんなカレーですか？（ビーフ？チキン？）」
*   「材料は何を使いますか？（じゃがいもは？人参は？）」
*   「どんな手順で作りますか？（先に肉を炒める？野菜から？）」

シェフを動かすには、もっと具体的で、正確な指示が必要です。そこで、あなたは「レシピ」を書くことにしました。

1.  **材料（データ）**: 牛肉 200g, じゃがいも 2個, 人参 1本, 玉ねぎ 1個, カレールー 1箱…
2.  **手順（コード）**:
    1.  玉ねぎを薄切りにする。
    2.  じゃがいもと人参の皮をむき、一口大に切る。
    3.  鍋に油をひき、玉ねぎを飴色になるまで炒める。
    4.  牛肉を加えて、色が変わるまで炒める。
    5.  …

この「**レシピ**」こそが、プログラミングの世界でいう「**プログラム**」です。そして、**このレシピを書く行為が「プログラミング」** なのです。

| 料理の世界 | プログラミングの世界 | 説明 |
| :--- | :--- | :--- |
| シェフ | コンピュータ | 指示通りに動く実行役 |
| 材料 | データ | プログラムが処理する情報 |
| レシピ | プログラム（ソースコード） | コンピュータへの指示書 |
| レシピを書くこと | プログラミング | 指示書を作成する行為 |
| 完成したカレー | アプリやWebサイト | プログラムが実行された結果 |

このように、プログラミングは決して特別な魔法ではありません。「コンピュータに正確に、かつ順番通りに指示を出す」という、非常に論理的な作業なのです。そして、このコースを終える頃には、あなたもこの「レシピ」を書けるようになっています。

## 概念の説明

### 🔑 プログラミングとは何か？

導入で「レシピ」の例え話をしましたが、もう少しだけ専門的な言葉で説明してみましょう。

> **プログラミング**とは、コンピュータが理解できる言葉（**プログラミング言語**）を使って、コンピュータに実行させたい処理の**手順（アルゴリズム）**を記述することです。

ここで新しい言葉が2つ出てきましたね。

1.  **プログラミング言語**: 人間とコンピュータがコミュニケーションするための「言葉」です。日本語や英語に様々な種類があるように、プログラミング言語にも`PHP`、`JavaScript`、`Python`など、たくさんの種類があります。このコースでは、主に`PHP`という言語を学びます。

2.  **アルゴリズム**: 「計算や問題解決のための手順」のことです。難しく聞こえますが、先ほどのカレーのレシピで言えば、「玉ねぎを炒めてから肉を入れる」といった「**処理の順番**」のことだと思ってください。

結局のところ、やっていることは同じです。「コンピュータがわかる言葉で、やってほしいことの順番を、正確に書く」。これがプログラミングの正体です。

### 🔑 なぜ今、プログラミングを学ぶのか？

「プログラミングが何かは分かったけど、なぜそれがそんなに重要なの？」と思うかもしれません。理由は大きく3つあります。

#### 1. あらゆる作業を「自動化」できるから

私たちの身の回りには、単純で退屈な「繰り返し作業」がたくさんあります。

*   毎月の売上データをExcelにまとめて、グラフを作る。
*   何百人もの顧客リストに、時候の挨拶メールを送る。
*   Webサイトから特定の情報を探し出して、一覧表にまとめる。

こうした作業を人間がやると、時間がかかる上に、ミスも起こりがちです。しかし、コンピュータは文句も言わず、間違えず、人間とは比べ物にならないスピードで作業をこなしてくれます。

プログラミングを学べば、こうした**「面倒な繰り返し作業」を自動化するプログラム**を作れるようになります。これにより、私たちはもっと創造的で、人間にしかできない仕事に集中できるようになるのです。

#### 2. 新しい「問題解決の道具」を作れるから

プログラミングは、単に既存の作業を効率化するだけではありません。世の中にある「こんなものがあったら便利なのに」というアイデアを、**自分自身の手で形にする力**を与えてくれます。

*   「毎日の勉強時間を記録して、グラフでやる気を管理したい」→ 勉強管理アプリを作ろう！
*   「地元の飲食店のテイクアウト情報をまとめたい」→ 地域密着のWebサイトを作ろう！
*   「チームのタスクを共有して、進捗を管理したい」→ タスク管理ツールを作ろう！

あなたが普段感じている不便や、「もっとこうだったら良いのに」という思いは、すべて新しいサービスやアプリの種になります。プログラミングは、その種を育て、花を咲かせるための「魔法の道具」なのです。

#### 3. 論理的思考力が身につくから

プログラミングは、コンピュータに「正確な指示」を与える作業だと説明しました。これは、物事を順序立てて、論理的に考える訓練になります。

*   目標を達成するには、どんな手順が必要か？（処理の分解）
*   もし、予期せぬ事態が起きたら、どう対処するか？（エラー処理）
*   どうすれば、もっと効率的に、無駄なく処理できるか？（最適化）

こうした**論理的思考力（ロジカルシンキング）**は、プログラミングだけでなく、日常生活や他の仕事においても非常に役立つ普遍的なスキルです。

## 詳細解説

### ⚙️ プログラミングで、具体的に何が作れるの？

プログラミングで作れるものは、今や私たちの生活のあらゆる場所に存在します。いくつか具体例を見てみましょう。

#### 1. Webサイト・Webアプリケーション

皆さんが毎日使っているであろう、WebサイトやWebサービスは、プログラミングによって作られています。

*   **Amazon**: 商品を検索し、カートに入れ、決済する。この一連の流れはすべてプログラムで制御されています。
*   **YouTube**: 動画をアップロードし、世界中の人がそれを見て、コメントを書き込む。これもプログラミングの力です。
*   **Twitter / Instagram**: 写真や文章を投稿し、「いいね」やフォローで繋がる。これも立派なWebアプリケーションです。

このコースで学ぶ`Laravel`という技術は、まさにこうしたWebアプリケーションを作るためのものです。

#### 2. スマートフォンアプリ

iPhoneやAndroidで使っているアプリも、プログラミングで作られています。

*   **LINE**: メッセージを送受信したり、スタンプを送ったりする機能。
*   **Googleマップ**: 現在地を取得し、目的地までの最適なルートを検索する機能。
*   **スマホゲーム**: キャラクターを動かしたり、敵と戦ったりする複雑な処理。

それぞれ、`Swift`（iPhoneアプリ用）や`Kotlin`（Androidアプリ用）といった、専用のプログラミング言語が使われています。

#### 3. 仕事で使うシステム（業務システム）

私たちの目には見えにくいですが、社会を支える多くのシステムもプログラミングで作られています。

*   **銀行のATM**: お金を引き出したり、振り込んだりするシステム。
*   **コンビニのレジ（POSシステム）**: 商品のバーコードを読み取り、在庫を管理するシステム。
*   **会社の勤怠管理システム**: 出社・退社時間を記録し、給与を計算するシステム。

これらがあるおかげで、私たちの社会はスムーズに動いているのです。

#### 4. ゲーム

家庭用ゲーム機やPCで遊ぶゲームも、プログラミングの塊です。
キャラクターの動き、グラフィックの表示、ストーリーの進行など、ゲームの世界を構成するあらゆる要素が、`C++`や`C#`といった言語で書かれたプログラムによって制御されています。

#### 5. AI（人工知能）やデータ分析

最近話題のAIも、プログラミングによって作られています。
膨大なデータを学習し、画像認識や自然言語処理といった高度な判断を行うプログラムです。`Python`という言語が、この分野では広く使われています。

## 💡 TIP: 学習を始める前の心構え

*   **完璧を目指さない**: 最初から全てを完璧に理解しようとすると、必ず挫折します。「今はなんとなく分かればOK」という気持ちで、どんどん先に進みましょう。後で戻ってきたときに、「ああ、こういうことだったのか！」と理解できる瞬間が必ず来ます。
*   **暗記ではなく、スキル**: プログラミングは、英単語のように全てを暗記するものではありません。自転車の乗り方と同じで、何度も転びながら、体で覚えていく「スキル」です。文法を忘れたら、その都度調べれば良いのです。
*   **楽しむことが一番**: 学習の過程で「これを作ってみたい！」という目標を見つけることが、モチベーションを維持する最大の秘訣です。このコースで学んだ知識を使って、自分だけの小さな作品を作ってみることをお勧めします。
