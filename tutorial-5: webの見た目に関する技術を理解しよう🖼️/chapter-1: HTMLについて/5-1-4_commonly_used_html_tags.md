# Tutorial 5-1-4: よく使うHTMLタグ

## 🎯 このセクションで学ぶこと

*   見出し、段落、リスト、強調など、Webページのコンテンツを構成するために、頻繁に使われる基本的なHTMLタグを学ぶ。
*   `<div>` と `<span>` という、汎用的なグループ化のためのタグの役割と、その違いを理解する。
*   タグの「入れ子（ネスト）」構造を理解する。

## 導入

前のセクションで、HTML文書の全体構造を学びました。今回は、いよいよ、`<body>` タグの中に、具体的なコンテンツを追加していくための、様々なタグを紹介します。

HTMLには、100種類以上のタグが存在しますが、日常的なWeb制作で使うタグは、そのうちの20〜30種類程度です。全てを一度に暗記する必要はありません。まずは、「**文章の構造を作る**」という観点から、最も基本的で、最もよく使うタグをマスターすることが重要です。

ここでは、それらのタグを、「**文房具セット**」に例えて紹介します。見出しを作るための「太いマジックペン」、段落を作るための「ボールペン」、リストを作るための「箇条書きマーカー」など、それぞれの文房具（タグ）が、どんな役割を持っているのかを、見ていきましょう。

---

## 詳細解説

### 📚 文章構造を作るタグ

#### 1. 見出しタグ: `<h1>` 〜 `<h6>`

*   **役割**: 文章の「見出し」や「タイトル」を示します。`<h1>` が最も重要度が高い大見出しで、数字が大きくなるにつれて、重要度が下がっていきます（`<h6>` が最も小さい小見出し）。
*   **使い方**: 本の章立てのように、文章の階層構造を、正しく表現するために使います。`<h1>` は、通常、1ページに1回だけ、そのページのメインタイトルとして使用します。
*   **注意点**: 文字を大きくしたい、という**見た目だけの理由で、見出しタグを使わない**でください。あくまで、その部分が、構造的に「見出し」である場合にのみ、使用します。見た目の調整は、CSSの役割です。

```html
<h1>第1章: HTMLの冒険</h1>
  <h2>1-1節: 冒険の始まり</h2>
    <h3>トピック: 最初のタグ</h3>
  <h2>1-2節: 新たな仲間</h2>
```

#### 2. 段落タグ: `<p>`

*   **役割**: 文章の「段落 (Paragraph)」を示します。最も基本的な、テキストブロックを作るためのタグです。
*   **使い方**: ブラウザは、`<p>` タグで囲まれた部分を、一つのまとまりとして認識し、通常、前後に、適度な余白を設けて表示します。

```html
<p>これは最初の段落です。HTMLは、Webページの骨格を作ります。</p>
<p>これは二つ目の段落です。CSSは、その見た目を装飾します。</p>
```

#### 3. 強調タグ: `<strong>` と `<em>`

*   **`<strong>`**: **重要性**が高い部分を示します。ブラウザは、通常、太字で表示します。
*   **`<em>`**: 文脈の中で、特に**強調**したい部分を示します。ブラウザは、通常、斜体（イタリック体）で表示します。

```html
<p>このチュートリアルで、<strong>最も重要な</strong>のは、継続することです。</p>
<p>これは、<em>絶対に</em> 忘れないでください。</p>
```

#### 4. リストタグ: `<ul>`, `<ol>`, `<li>`

*   **役割**: 項目を「リスト（一覧）」形式で表示します。
*   **`<ul>` (Unordered List)**: 順序のない、箇条書きリストを作ります。通常、点（・）や丸（●）が、項目の頭に付きます。
*   **`<ol>` (Ordered List)**: 1, 2, 3... のような、順序のあるリストを作ります。
*   **`<li>` (List Item)**: `<ul>` または `<ol>` の中で使われ、個々のリスト項目を示します。

```html
<!-- 順序のないリスト -->
<ul>
  <li>りんご</li>
  <li>ばなな</li>
  <li>みかん</li>
</ul>

<!-- 序のあるリスト -->
<ol>
  <li>HTMLファイルを準備する</li>
  <li>コードを書く</li>
  <li>ブラウザで確認する</li>
</ol>
```

### 📦 グループ化のためのタグ

ここまでに紹介したタグは、それぞれ「見出し」「段落」といった、具体的な意味を持っていました。しかし、時には、**特に意味はないけれど、いくつかの要素を、ひとまとめにして扱いたい**、という場合があります。例えば、レイアウトの都合で、複数の段落や画像を、一つの箱に入れたい、といったケースです。

そのために使われるのが、`<div>` と `<span>` という、汎用的なコンテナタグです。

#### 1. `<div>` タグ (Division)

*   **役割**: 「分割」や「区分」を意味し、複数の要素をグループ化するための、**ブロックレベル**のコンテナです。
*   **特徴**: `<div>` タグ自体は、特に意味を持ちませんが、CSSでスタイルを適用したり、JavaScriptで操作したりするための「箱」として機能します。`<div>` で囲まれた要素は、通常、前後に改行が入り、横幅いっぱいに広がる、一つのブロックとして扱われます。

```html
<div style="background-color: #f0f0f0; padding: 15px;">
  <h2>プロフィール</h2>
  <p>名前: 山田 太郎</p>
  <p>趣味: プログラミング</p>
</div>
```

#### 2. `<span>` タグ

*   **役割**: `<div>` と同じく、グループ化のためのタグですが、こちらは、文章の一部など、より小さい範囲を対象とする、**インラインレベル**のコンテナです。
*   **特徴**: `<span>` で囲んでも、前後に改行は入りません。文章の流れを断ち切ることなく、特定の部分だけを、グループ化することができます。例えば、文章中の一部の単語だけ、色を変えたい、といった場合に使います。

```html
<p>私の好きな色は、<span style="color: red;">赤色</span>です。</p>
```

> **ブロック vs インライン**: HTMLの要素は、大きく分けて「ブロックレベル要素」と「インラインレベル要素」に分類されます。`<h1>` や `<p>`, `<div>` は、一つのブロックとして扱われ、前後に改行が入ります。一方、`<a>` や `<strong>`, `<span>` は、文章の一部として扱われ、改行は入りません。この違いは、レイアウトを組む上で、非常に重要になります。

## ✨ まとめ

このセクションでは、Webページのコンテンツを構成する、基本的なHTMLタグについて学びました。

| タグ | 名称 | レベル | 役割 |
| :--- | :--- | :--- | :--- |
| `<h1>`~`<h6>` | 見出し | ブロック | 文章の階層構造を作る |
| `<p>` | 段落 | ブロック | テキストのまとまりを作る |
| `<strong>` | 重要 | インライン | テキストの重要性を示す（太字） |
| `<em>` | 強調 | インライン | テキストを強調する（斜体） |
| `<ul>`, `<ol>` | リスト | ブロック | 箇条書きや番号付きリストを作る |
| `<li>` | リスト項目 | ブロック | リストの各項目を作る |
| `<div>` | 分割 | ブロック | 複数の要素をまとめる「大きな箱」 |
| `<span>` | 範囲 | インライン | 文章の一部をまとめる「小さな箱」 |

これらの基本的な「文房具」を使いこなすことで、あなたは、構造化された、意味のあるWebページを、自由に作成することができます。
