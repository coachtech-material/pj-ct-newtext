# Tutorial 5-2-3: セレクタの種類

## 🎯 このセクションで学ぶこと

*   よく使われる基本的なセレクタの種類を理解する。
*   CSSのコードを読んだときに、「どの要素にスタイルが適用されているか」がわかるようになる。

---

## 詳細解説

セレクタは、「どのHTML要素にスタイルを適用するか」を指定するものです。ここでは、最もよく使われる基本的なセレクタを紹介します。

### 1. タグセレクタ（要素セレクタ）

HTMLのタグ名を、そのまま指定します。指定したタグ**全て**に、スタイルが適用されます。

```css
/* 全ての<p>タグに適用 */
p {
  color: gray;
}
```

### 2. クラスセレクタ

HTMLの `class` 属性で指定したクラス名を使って、要素を選択します。クラス名の前に、ドット `.` を付けます。**最もよく使われるセレクタ**です。

```html
<p class="highlight">この段落は強調されます</p>
<p>この段落は通常のままです</p>
```

```css
/* class="highlight" を持つ要素に適用 */
.highlight {
  background-color: yellow;
}
```

> 📝 **ポイント**: 同じクラス名を、複数の要素に付けることができます。共通のスタイルを適用したい要素に、同じクラス名を付けるのが、CSSの基本的な使い方です。

### 3. IDセレクタ

HTMLの `id` 属性で指定したID名を使って、要素を選択します。ID名の前に、シャープ `#` を付けます。

```html
<div id="header">ヘッダー</div>
```

```css
/* id="header" を持つ要素に適用 */
#header {
  background-color: navy;
  color: white;
}
```

> 📝 **ポイント**: IDは、1つのHTMLファイル内で**一意（ユニーク）**でなければなりません。同じIDを、複数の要素に付けることはできません。そのため、スタイル適用にはクラスセレクタを使い、IDはJavaScriptからの操作などに使うことが多いです。

---

## ✨ まとめ

| セレクタ | 書き方 | 適用対象 |
| :--- | :--- | :--- |
| **タグセレクタ** | `p { }` | 指定したタグ全て |
| **クラスセレクタ** | `.クラス名 { }` | 指定したクラスを持つ要素（複数可） |
| **IDセレクタ** | `#ID名 { }` | 指定したIDを持つ要素（1つのみ） |

実際の開発では、**クラスセレクタ**が最も多く使われます。CSSのコードを読むときは、`.`（クラス）と `#`（ID）の違いを意識してみてください。
