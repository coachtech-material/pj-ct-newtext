# Tutorial 5-3-2: メディアクエリの使い方

## 🎯 このセクションで学ぶこと

*   メディアクエリの基本的な構文を理解する。
*   `max-width` と `min-width` の意味を知る。

---

## 詳細解説

### 📜 メディアクエリの基本構文

メディアクエリは、`@media` というキーワードで始まり、条件を括弧 `()` で囲んで記述します。

```css
@media (条件) {
  /* 条件が真のときに適用されるCSSルール */
  セレクタ {
    プロパティ: 値;
  }
}
```

### 📐 条件の指定: `max-width` と `min-width`

条件として最もよく使われるのが、画面幅に関するものです。

| 条件 | 意味 | 例 |
| :--- | :--- | :--- |
| `max-width: 768px` | 画面幅が**768px以下**の場合 | スマホ・タブレット向け |
| `min-width: 769px` | 画面幅が**769px以上**の場合 | PC向け |

```css
/* 基本のスタイル */
.container {
  display: flex;  /* 横並び */
}

/* 画面幅が768px以下になったら */
@media (max-width: 768px) {
  .container {
    display: block;  /* 縦並びに戻す */
  }
}
```

### 📝 ブレークポイント

スタイルを切り替える画面幅のことを、**ブレークポイント (Breakpoint)** と呼びます。

一般的によく使われる目安：

| 画面幅 | 想定デバイス |
| :--- | :--- |
| ~ 600px | スマートフォン |
| 600px ~ 960px | タブレット |
| 960px ~ | PC |

---

## ✨ まとめ

*   **メディアクエリ**: `@media (条件) { }` の形で、条件に応じたスタイルを記述する
*   **max-width**: 「〇〇px**以下**」の場合に適用
*   **min-width**: 「〇〇px**以上**」の場合に適用
*   **ブレークポイント**: スタイルを切り替える画面幅のこと

CSSのコードで `@media (max-width: 768px)` のような記述を見かけたら、「768px以下（スマホ・タブレット）の場合のスタイル」と読み替えられれば十分です。
