# Tutorial 7-1-1: PHPとは何か

## 🎯 このセクションで学ぶこと

*   PHPがどのようなプログラミング言語なのか、その特徴と歴史を理解する。
*   サーバーサイド言語とクライアントサイド言語の違いを明確に区別できるようになる。
*   なぜ今、数ある言語の中からPHP（そしてLaravel）を学ぶのか、その理由を理解する。

---

## 導入

HTMLとCSSを学び、Webページの骨格と見た目を作れるようになりました。しかし、それだけでは「静的な」ページしか作れません。つまり、いつ誰が見ても同じ内容が表示されるだけの、動きのないページです。

今日のWebサイトのほとんどは、ユーザーの操作に応じて表示内容が変わったり、データベースと連携して新しい情報を表示したりする「動的な」ページで構成されています。例えば、ブログの記事一覧、ECサイトの商品リスト、SNSのタイムラインなど、これらは全て動的なページです。

このような動的なWebサイトを構築するために使われるのが、**サーバーサイド言語**です。そして、そのサーバーサイド言語の中で、最も広く使われ、長い歴史を持つ言語の一つが、今回から学ぶ**PHP (PHP: Hypertext Preprocessor)** なのです。

このセクションでは、PHPがどのような言語で、Webの世界でどのような役割を果たしているのか、その全体像を掴んでいきましょう。

---

## 詳細解説

### 📜 PHPとは？

PHPは、**動的なWebページを作成することに特化した、サーバーサイドのスクリプト言語**です。

*   **サーバーサイド**: Webサーバー側でプログラムが実行される、という意味です。後述するクライアントサイド言語（JavaScript）と対比されます。
*   **スクリプト言語**: コンパイル（機械語への事前翻訳）が不要で、コードを記述したらすぐに実行できる手軽な言語のことです。

PHPの最大の特徴は、**HTMLに直接埋め込んで記述できる**という点です。これにより、HTMLの構造の中に、動的な要素を簡単に追加することができます。

```php
<!DOCTYPE html>
<html>
<head>
    <title>PHPのテスト</title>
</head>
<body>
    <h1>こんにちは！</h1>
    <p>現在の日時は、<?php echo date("Y年m月d日 H時i分s秒"); ?> です。</p>
</body>
</html>
```

この例では、`<?php ... ?>`というタグで囲まれた部分がPHPのコードです。`date()`関数を使って現在の日時を取得し、`echo`でその結果をHTMLの中に「出力」しています。このPHPファイルにアクセスすると、サーバーがPHPコードを実行し、その結果（現在の日時）を含んだ、完全なHTMLを生成して、ブラウザに返します。

### サーバーサイド vs クライアントサイド

Web開発で使われる言語は、実行される場所によって、**サーバーサイド言語**と**クライアントサイド言語**の2つに大別されます。この違いを理解することは非常に重要です。

| 項目 | サーバーサイド言語 | クライアントサイド言語 |
| :--- | :--- | :--- |
| **代表的な言語** | PHP, Ruby, Python, Java, Go | JavaScript |
| **実行される場所** | Webサーバー | Webブラウザ（ユーザーのPC） |
| **主な役割** | ・データベースとの連携<br>・ユーザー認証<br>・HTMLの動的な生成<br>・ビジネスロジックの実行 | ・HTML/CSSの操作（アニメーションなど）<br>・ユーザー入力の検証<br>・サーバーとの非同期通信（Ajax） |
| **比喩** | レストランの**キッチン**（調理担当） | レストランの**ホール**（接客・配膳担当） |

サーバーサイド（PHP）は、ユーザーの目に見えない「裏側」で、データベースからデータを取得したり、注文を処理したりといった、**本質的な処理**を担当します。一方、クライアントサイド（JavaScript）は、ユーザーが直接触れる「表側」で、ページの見た目をリッチにしたり、使いやすくしたりする**演出**を担当します。

Laravelは、このサーバーサイドの処理を、効率的かつ安全に開発するための、PHPのフレームワークなのです。

### なぜ今、PHP（Laravel）を学ぶのか？

「PHPは古い」という声を聞くことがあるかもしれません。確かに、PHPは1995年に登場した、比較的歴史の長い言語です。[1] しかし、今なお、PHPがWeb開発の第一線で活躍し続けているのには、明確な理由があります。

1.  **圧倒的なシェア**: 全世界のWebサイトのうち、実に**77%以上**が、サーバーサイドにPHPを使用しています。[2] 特に、ブログやメディアサイトで絶大な人気を誇るCMS（コンテンツ管理システム）である**WordPress**がPHPで作られていることが、この高いシェアを支えています。

2.  **モダンな進化**: 「古い」と言われたPHPは、バージョン7以降、劇的なパフォーマンス向上と、モダンな言語機能の導入を果たしました。現在のPHP 8は、他の言語と比較しても遜色のない、強力で安全な言語に進化しています。

3.  **Laravelの存在**: そして、PHPを学ぶ最大の理由の一つが、**Laravel**という、非常に優れたフレームワークの存在です。Laravelは、「Web職人のためのPHPフレームワーク」を標榜し、開発者が楽しく、効率的に、高品質なアプリケーションを構築するための、あらゆる機能を提供しています。その美しい設計と、豊富な機能により、世界中のPHPコミュニティで、最も人気のあるフレームワークとしての地位を確立しています。

4.  **豊富な求人数と学習リソース**: 長い歴史と高いシェアのおかげで、PHP/Laravelに関する求人数は非常に多く、また、Web上のドキュメントや学習リソースも豊富に存在します。これは、初学者が学習を進め、キャリアを築いていく上で、大きなアドバンテージとなります。

このカリキュラムでは、モダンに進化したPHPの基礎を学び、それを土台として、Laravelという強力な武器を使いこなすことを目指します。

## ✨ まとめ

このセクションでは、PHPがどのような言語で、なぜ今それを学ぶのかについて、その全体像を掴みました。

*   **PHP**: 動的なWebページを作成することに特化した、**サーバーサイド**のスクリプト言語。
*   **サーバーサイドとクライアントサイド**: サーバーサイド（PHP）は**裏側の処理**（データベース連携など）、クライアントサイド（JavaScript）は**表側の演出**（アニメーションなど）を担当する。
*   **PHPを学ぶ理由**: **圧倒的なシェア**、**モダンな進化**、そして何よりも**Laravelという最高のフレームワーク**の存在が、PHPを学ぶ強力な動機となる。

PHPは、Web開発の歴史そのものであり、今なお、その中心にあり続ける、パワフルな言語です。次のセクションから、いよいよ、その文法と、具体的な書き方を学んでいきましょう。
